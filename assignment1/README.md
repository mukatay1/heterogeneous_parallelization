В данных работах реализованы алгоритмы на C++ для обработки больших массивов данных с использованием динамического управления памятью и технологий параллельного программирования OpenMP. Основной упор сделан на сравнение производительности последовательных и параллельных методов при выполнении поиска экстремумов и вычисления статистических показателей.
