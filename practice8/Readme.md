Контрольные вопросы:

1.Преимущества гибридных вычислений заключаются в более эффективном использовании ресурсов CPU и GPU и повышении общей производительности.

2.Накладные расходы при передаче данных минимизируются за счёт уменьшения числа копирований и оптимального распределения вычислений.

3.На CPU лучше выполнять управляющие и последовательные задачи, а на GPU — массовые параллельные вычисления.

4.Производительность гибридного приложения можно улучшить с помощью балансировки нагрузки и оптимизации передачи данных между CPU и GPU.
